# OrangeHat Lead Recovery Kit - Quick Start

Welcome. This kit is designed to help local service businesses recover missed revenue from existing leads in under 24 hours.

## What to do first (30-60 min)

1. If you're non-technical, start with `00A-how-to-use-this-kit-non-technical.md`
2. Pick your booking destination (calendar link / call scheduler)
3. Add your business variables in `02-implementation-sop.md`
4. Launch 3 flows in this order:
   - Missed-call auto-reply
   - 7-day follow-up sequence
   - Old-lead reactivation campaign
5. If needed, import contacts using the matching CRM file in `/imports`
6. Track results in `08-kpi-tracker.csv`
