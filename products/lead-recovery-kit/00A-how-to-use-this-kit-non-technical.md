## What is a `.md` file?
A `.md` (Markdown) file is like a normal note or Word document, but plain text.
You can open and edit it with common apps.

## Easiest way to open these files

### On Windows
- Right-click the file
- Choose **Open with**
- Select **Notepad** (or WordPad, VS Code, Notion, Typora)

### On Mac
- Double-click (usually opens in TextEdit), or
- Right-click -> **Open With** -> TextEdit

### On Phone
- Use Google Docs, Apple Notes, or any text editor app
- You can also email files to yourself and open from there

## How to use the scripts quickly
1. Open a script file (example: `01-missed-call-sms-scripts.md`)
2. Copy the template message
3. Replace placeholders like:
   - `{{business_name}}`
   - `{{booking_link}}`
   - `{{phone_number}}`
4. Paste into your CRM/SMS platform
